# Task 1: Group Discussion Preparation Workshop Landing Page
A professional static landing page designed for a Group Discussion Preparation Workshop. This project focuses on creating an engaging event website with a strong visual presence and clear user flow.

## Features
- Semantic HTML5 structure
- Workshop introduction and event details
- Registration form section
- Custom CSS styling
- Responsive layout design
- Professional typography and color palette

## Technologies Used
- HTML5
- CSS3
- Google Fonts

## Project Objective
To build a clean and visually appealing static webpage while practicing fundamental frontend development concepts including layout structuring, styling, positioning, and user-focused design.
